Basic If Statement</div> <div class="text">The basic syntax of an if statement is:</div> <div class="text"> if condition: <br> # code to execute if the condition is True </div> <div class="text">Example:</div> <div class="text"> x = 10<br> if x > 5: <br> print("x is greater than 5") </div> <div class="sub-header">2. If-Else Statement</div> <div class="text">You can use else to specify the code to execute when the condition is False:</div> <div class="text"> if condition: <br> # code to execute if the condition is True <br> else: <br> # code to execute if the condition is False </div> <div class="text">Example:</div> <div class="text"> y = 3<br> if y > 5: <br> print("y is greater than 5") <br> else: <br> print("y is not greater than 5") </div> <div class="sub-header">3. If-Elif-Else Statement</div> <div class="text">You can use elif (short for "else if") to check multiple conditions:</div> <div class="text"> if condition1: <br> # code to execute if condition1 is True <br> elif condition2: <br> # code to execute if condition2 is True <br> else: <br> # code to execute if all conditions are False </div> <div class="text">Example:</div> <div class="text"> z = 7<br> if z < 5: <br> print("z is less than 5") <br> elif z == 5: <br> print("z is equal to 5") <br> else: <br> print("z is greater than 5") </div> <div class="header">Conclusion</div> <div class="text">If statements are fundamental for controlling the flow of your Python programs.
