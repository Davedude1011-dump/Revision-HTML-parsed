Addressing and Pointers</div> <div class="text">- **Address Registers:** Some registers store memory addresses, facilitating data retrieval and storage operations.</div> <div class="text">- **Pointer Manipulation:** Registers play a role in managing pointers and facilitating efficient memory access.</div> <div class="header">5. Parallel Processing and Pipelining</div> <div class="text">- **Parallel Access:** Modern CPUs often support parallel access to multiple registers, enabling simultaneous execution of instructions.</div> <div class="text">- **Pipeline Stages:** Registers are used to store data in different stages of the instruction pipeline, facilitating efficient pipelining.</div> <div class="header">7.
